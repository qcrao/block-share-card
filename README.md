# Block Share Card Extension for Roam Research

The Block Share Card extension for Roam Research enables users to share content blocks as beautiful card images with multiple theme options.

## Card Styles

| Classic | Modern Light | Modern Dark |
| ------- | ------------ | ----------- |
| ![Classic](https://github.com/qcrao/block-share-card/blob/main/assets/classic.png?raw=true) | ![Modern Light](https://github.com/qcrao/block-share-card/blob/main/assets/modern-light.png?raw=true) | ![Modern Dark](https://github.com/qcrao/block-share-card/blob/main/assets/modern-dark.png?raw=true) |

## Features

- **Three Card Styles:** Choose from Classic, Modern Light, and Modern Dark themes.
- **Multiple Block Selection:** Share a single block or select multiple blocks at once.
- **Flexible Triggering:** Use the toolbar button or the Command Palette to generate card images.
- **No Zoom Required:** Works with zoomed-in blocks, focused blocks, and selected blocks.

## Installation

1. Open Roam Research and navigate to Roam Depot.
2. Search for "Block Share Card" in Roam Depot's search bar.
3. Click the "Install" button to add the extension.

## Usage

### Method 1: Toolbar Button

1. Zoom into a block, click on a block, or select multiple blocks.

| Zoom into a block | Select multiple blocks |
| --- | --- |
| ![](https://github.com/qcrao/block-share-card/blob/main/assets/step1.jpeg?raw=true) | ![](https://github.com/qcrao/block-share-card/blob/main/assets/1-select-multi-blocks.png?raw=true) |

2. Click the share button in the toolbar and choose a card style.

![](https://github.com/qcrao/block-share-card/blob/main/assets/2-click-the-button.png?raw=true)

3. The image will be automatically downloaded.

### Method 2: Command Palette

1. Zoom into a block, click on a block, or select multiple blocks.
2. Open the Command Palette and search for "Share Block Card".
3. Select a style: Classic, Modern Light, or Modern Dark.

![](https://github.com/qcrao/block-share-card/blob/main/assets/command-platte.png?raw=true)
